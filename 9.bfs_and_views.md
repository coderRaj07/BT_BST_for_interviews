## 1️⃣ Level Order Traversal

🔗 [https://leetcode.com/problems/binary-tree-level-order-traversal/](https://leetcode.com/problems/binary-tree-level-order-traversal/)

### 🔹 Trick

Standard BFS.
Push children left → right.

```python
class Solution:
    def levelOrder(self, root):
        if not root:
            return []
        
        from collections import deque
        q = deque([root])
        res = []
        
        while q:
            level = []
            for _ in range(len(q)):
                node = q.popleft()
                level.append(node.val)
                
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)
            
            res.append(level)
        
        return res
```

## 2️⃣ Zigzag Level Order

🔗 [https://leetcode.com/problems/binary-tree-zigzag-level-order-traversal/](https://leetcode.com/problems/binary-tree-zigzag-level-order-traversal/)

### 🔹 Trick

Alternate direction every level.

```python
class Solution:
    def zigzagLevelOrder(self, root):
        if not root:
            return []
        
        from collections import deque
        q = deque([root])
        res = []
        left_to_right = True
        
        while q:
            level = []
            for _ in range(len(q)):
                node = q.popleft()
                level.append(node.val)
                
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)
            
            if not left_to_right:
                level.reverse()
            
            res.append(level)
            left_to_right = not left_to_right
        
        return res
```

## 3️⃣ Right View

🔗 [https://leetcode.com/problems/binary-tree-right-side-view/](https://leetcode.com/problems/binary-tree-right-side-view/)

### 🔹 Trick

At each level, take last node.

```python
class Solution:
    def rightSideView(self, root):
        if not root:
            return []
        
        from collections import deque
        q = deque([root])
        res = []
        
        while q:
            size = len(q)
            for i in range(size):
                node = q.popleft()
                
                if i == size - 1:
                    res.append(node.val)
                
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)
        
        return res
```

## 5️⃣ Top View

🔗 [https://practice.geeksforgeeks.org/problems/top-view-of-binary-tree/1](https://practice.geeksforgeeks.org/problems/top-view-of-binary-tree/1)

### 🔹 Trick

Track horizontal distance (hd).
Store first node per hd.

```python
def topView(root):
    from collections import deque
    if not root:
        return []
    
    q = deque([(root, 0)])
    hd_map = {}
    
    while q:
        node, hd = q.popleft()
        
        if hd not in hd_map:
            hd_map[hd] = node.data
        
        if node.left:
            q.append((node.left, hd - 1))
        if node.right:
            q.append((node.right, hd + 1))
    
    return [hd_map[x] for x in sorted(hd_map)]
```

## 7️⃣ Vertical Order Traversal

🔗 [https://leetcode.com/problems/vertical-order-traversal-of-a-binary-tree/](https://leetcode.com/problems/vertical-order-traversal-of-a-binary-tree/)

### 🔹 Trick

Track:

* horizontal distance
* level
  Sort properly.

```python
class Solution:
    def verticalTraversal(self, root):
        from collections import defaultdict, deque
        
        nodes = []
        q = deque([(root, 0, 0)])  # node, row, col
        
        while q:
            node, row, col = q.popleft()
            if node:
                nodes.append((col, row, node.val))
                q.append((node.left, row+1, col-1))
                q.append((node.right, row+1, col+1))
        
        nodes.sort()
        
        res = []
        prev_col = float('-inf')
        
        for col, row, val in nodes:
            if col != prev_col:
                res.append([])
                prev_col = col
            res[-1].append(val)
        
        return res
```

Now we move to build trees from traversals.

Core concept:


## 1️⃣ Construct BT from Preorder + Inorder

🔗 [https://leetcode.com/problems/construct-binary-tree-from-preorder-and-inorder-traversal/](https://leetcode.com/problems/construct-binary-tree-from-preorder-and-inorder-traversal/)

### 🔹 Trick

Preorder gives root.
Inorder splits left and right.

```python
class Solution:
    def buildTree(self, preorder, inorder):
        if not preorder or not inorder:
            return None
        
        root_val = preorder[0]
        root = TreeNode(root_val)
        
        mid = inorder.index(root_val)
        
        root.left = self.buildTree(preorder[1:mid+1], inorder[:mid])
        root.right = self.buildTree(preorder[mid+1:], inorder[mid+1:])
        
        return root
```
