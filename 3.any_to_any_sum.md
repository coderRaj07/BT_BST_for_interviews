Now we enter the most powerful pattern in BT.

This is where most Amazon / FAANG hard problems live.

# 🟦 SECTION 3 — ANY → ANY (GLOBAL UPDATE PATTERN)

This is NOT strictly root→leaf or leaf→leaf.

Path can start and end anywhere.

---

## ⭐ CORE FUNDAMENTAL RULE

```python id="s3rule1"
l = max(0, dfs(root.left))
r = max(0, dfs(root.right))

# update using BOTH
self.ans = max(self.ans, l + r + root.val)

# return only ONE branch upward
return root.val + max(l, r)
```

Key Understanding:

* Update uses BOTH branches
* Return uses ONE branch
* Ignore negative paths
* Postorder traversal required

# 1️⃣ Binary Tree Maximum Path Sum (Leetcode 124)

🔗 [https://leetcode.com/problems/binary-tree-maximum-path-sum/](https://leetcode.com/problems/binary-tree-maximum-path-sum/)

Type: Max Sum (Any → Any)

---

### 🔹 Trick

Ignore negative paths using max(0, dfs(...))

Update using l + r + root

---

### 🔹 Solution (Your Version)

```python
class Solution:
    def maxPathSum(self, node: Optional[TreeNode]) -> int:
        self.maxi = float('-inf')
        def dfs(root):
            if not root:
                return 0

            if not root.left and not root.right:
                self.maxi = max(self.maxi, root.val)
                return root.val
            
            l = max(0,dfs(root.left))
            r = max(0,dfs(root.right))

            self.maxi = max(self.maxi, l+r+root.val)

            return root.val+max(l,r)

        dfs(node)
        return self.maxi
```

# 2️⃣ Largest Subtree Sum (GFG)

🔗 [https://practice.geeksforgeeks.org/problems/largest-subtree-sum-in-a-tree/1](https://practice.geeksforgeeks.org/problems/largest-subtree-sum-in-a-tree/1)

Type: Any → Any (Subtree Aggregation)

---

### 🔹 Trick

Return subtree total.
Update global max using subtree total.

---

### 🔹 Solution

```python
class Solution:
    def findLargestSubtreeSum(self, root):
        self.maxi = float('-inf')

        def dfs(node):
            if not node:
                return 0
            
            l = dfs(node.left)
            r = dfs(node.right)

            total = node.data + l + r
            self.maxi = max(self.maxi, total)

            return total
        
        dfs(root)
        return self.maxi
```

# 3️⃣ Maximum Average Subtree (Leetcode 1120)

🔗 [https://leetcode.com/problems/maximum-average-subtree/](https://leetcode.com/problems/maximum-average-subtree/)

Type: Any → Any (Subtree return multiple values)

---

### 🔹 Trick

Return (sum, count)
Update using sum / count

---

### 🔹 Solution

```python
class Solution:
    def maximumAverageSubtree(self, root: Optional[TreeNode]) -> float:
        self.maxi = 0
        
        def dfs(node):
            if not node:
                return (0, 0)
            
            l_sum, l_count = dfs(node.left)
            r_sum, r_count = dfs(node.right)
            
            total_sum = node.val + l_sum + r_sum
            total_count = 1 + l_count + r_count
            
            self.maxi = max(self.maxi, total_sum / total_count)
            
            return (total_sum, total_count)
        
        dfs(root)
        return self.maxi
```

Return two values:

### 🔹 Solution

```python
class Solution:
    def rob(self, root: Optional[TreeNode]) -> int:
        
        def dfs(node):
            if not node:
                return (0, 0)
            
            l = dfs(node.left)
            r = dfs(node.right)
            
            rob_node = node.val + l[1] + r[1]
            not_rob = max(l) + max(r)
            
            return (rob_node, not_rob)
        
        return max(dfs(root))
```

# 5️⃣ Count Nodes Equal to Average of Subtree (Leetcode 2265)

🔗 [https://leetcode.com/problems/count-nodes-equal-to-average-of-subtree/](https://leetcode.com/problems/count-nodes-equal-to-average-of-subtree/)

Type: Any → Any (Subtree return sum & count)

---

### 🔹 Trick

Return (sum, count)
Compare node.val with sum // count

---

### 🔹 Solution

```python
class Solution:
    def averageOfSubtree(self, root: Optional[TreeNode]) -> int:
        self.count = 0
        
        def dfs(node):
            if not node:
                return (0, 0)
            
            l_sum, l_count = dfs(node.left)
            r_sum, r_count = dfs(node.right)
            
            total_sum = node.val + l_sum + r_sum
            total_count = 1 + l_count + r_count
            
            if node.val == total_sum // total_count:
                self.count += 1
            
            return (total_sum, total_count)
        
        dfs(root)
        return self.count
```
