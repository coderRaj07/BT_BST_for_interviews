# 🟦 SECTION 2 — LEAF → LEAF PATTERN

---

## ⭐ CORE FUNDAMENTAL RULE

```python
if root.left and root.right:
    # update answer here
```

Key difference from Section 1:

* Root → Leaf → choose ONE branch
* Leaf → Leaf → combine BOTH branches
* Update only when BOTH children exist
* Return only one branch upward

Return rule:

```python
return root.val + max(left, right)
```

# 1️⃣ Maximum Path Sum Leaf → Leaf (GFG)

🔗 [https://practice.geeksforgeeks.org/problems/maximum-path-sum/1](https://practice.geeksforgeeks.org/problems/maximum-path-sum/1)

### 🔹 Trick

Update only when both children exist.

---

### 🔹 Solution (Your Version)

```python
class Solution:        
    def maxPathSum(self, root):
        self.maxi = float('-inf')
        def dfs(node):
            if not node:
                return float('-inf')
            
            if not node.left and not node.right:
                return node.data
            
            l = dfs(node.left)
            r = dfs(node.right)
            
            if node.left and node.right:
                self.maxi = max(self.maxi, node.data+l+r)
                return node.data+max(l,r)
            
            return node.data+(l if node.left else r)
    
        total = dfs(root)
        
        if self.maxi == float('-inf'):
            return total
        
        return self.maxi
```

# 2️⃣ Diameter of Binary Tree (Leetcode 543)

🔗 [https://leetcode.com/problems/diameter-of-binary-tree/](https://leetcode.com/problems/diameter-of-binary-tree/)

### 🔹 Trick

Leaf → Leaf but measuring height instead of sum.
Update using:

```python
left_height + right_height
```

Return height.

---

### 🔹 Solution

```python
class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        self.maxi = 0
        
        def dfs(node):
            if not node:
                return 0
            
            l = dfs(node.left)
            r = dfs(node.right)
            
            self.maxi = max(self.maxi, l + r)
            
            return 1 + max(l, r)
        
        dfs(root)
        return self.maxi
```

# 3️⃣ Longest Univalue Path (Leetcode 687)

🔗 [https://leetcode.com/problems/longest-univalue-path/](https://leetcode.com/problems/longest-univalue-path/)

### 🔹 Trick

Same as diameter, but only extend branch if values match.

Update using both matching branches.

---

### 🔹 Solution

```python
class Solution:
    def longestUnivaluePath(self, root: Optional[TreeNode]) -> int:
        self.maxi = 0
        
        def dfs(node):
            if not node:
                return 0
            
            l = dfs(node.left)
            r = dfs(node.right)
            
            left_path = right_path = 0
            
            if node.left and node.left.val == node.val:
                left_path = l + 1
            if node.right and node.right.val == node.val:
                right_path = r + 1
            
            self.maxi = max(self.maxi, left_path + right_path)
            
            return max(left_path, right_path)
        
        dfs(root)
        return self.maxi
```

# 4️⃣ Binary Tree Cameras (Leetcode 968)

🔗 [https://leetcode.com/problems/binary-tree-cameras/](https://leetcode.com/problems/binary-tree-cameras/)

### 🔹 Trick

Postorder decision.
Leaf forces parent camera.

States:

* 0 → needs camera
* 1 → has camera
* 2 → covered

---

### 🔹 Solution

```python
class Solution:
    def minCameraCover(self, root: Optional[TreeNode]) -> int:
        self.cameras = 0
        
        def dfs(node):
            if not node:
                return 2
            
            l = dfs(node.left)
            r = dfs(node.right)
            
            if l == 0 or r == 0:
                self.cameras += 1
                return 1
            
            if l == 1 or r == 1:
                return 2
            
            return 0
        
        if dfs(root) == 0:
            self.cameras += 1
        
        return self.cameras
```
