# 🟦 SECTION 4 — PREFIX / SUM = K (ANY → ANY PATH COUNTING)

This section is different from Section 3.

Here:

* Path must go downward.
* Path can start anywhere.
* We COUNT paths.
* We use prefix sum hashmap.

---

## ⭐ CORE FUNDAMENTAL RULE

```python
curr_sum += node.val

count += prefix[curr_sum - k]

prefix[curr_sum] += 1

dfs(left)
dfs(right)

prefix[curr_sum] -= 1   # backtrack
```

Key Understanding:

* Path is downward only.
* prefix[x] means how many times sum x appeared before.
* curr_sum - k tells how many valid starting points exist.
* Backtracking is mandatory.

# 1️⃣ Path Sum III (Leetcode 437)

🔗 [https://leetcode.com/problems/path-sum-iii/](https://leetcode.com/problems/path-sum-iii/)

Type: Sum = K (Any → Any)

---

### 🔹 Trick

Use prefix hashmap.

Initialize:

```python
prefix[0] = 1
```

---

### 🔹 Solution (Your Version)

```python
class Solution:
    def pathSum(self, root: Optional[TreeNode], k: int) -> int:
        self.count = 0
        self.prefix = defaultdict(int)
        self.prefix[0] = 1

        def dfs(node, curr_sum):
            if not node:
                return 0

            curr_sum += node.val
            self.count += self.prefix[curr_sum - k]
            self.prefix[curr_sum] += 1

            dfs(node.left, curr_sum)
            dfs(node.right, curr_sum)

            self.prefix[curr_sum] -= 1

        dfs(root, 0)
        return self.count
```

# 2️⃣ Count Paths Sum = K (GFG Variant)

🔗 [https://practice.geeksforgeeks.org/problems/k-sum-paths/1](https://practice.geeksforgeeks.org/problems/k-sum-paths/1)

Type: Sum = K (Any → Any)

---

### 🔹 Trick

Same prefix idea.

---

### 🔹 Solution

```python
class Solution:
    def sumK(self, root, k):
        from collections import defaultdict
        
        prefix = defaultdict(int)
        prefix[0] = 1
        self.count = 0
        
        def dfs(node, curr_sum):
            if not node:
                return
            
            curr_sum += node.data
            self.count += prefix[curr_sum - k]
            prefix[curr_sum] += 1
            
            dfs(node.left, curr_sum)
            dfs(node.right, curr_sum)
            
            prefix[curr_sum] -= 1
        
        dfs(root, 0)
        return self.count
```

# 3️⃣ Subtree With Given Sum (GFG)

🔗 [https://practice.geeksforgeeks.org/problems/subtree-with-given-sum/1](https://practice.geeksforgeeks.org/problems/subtree-with-given-sum/1)

Type: Subtree sum check

---

### 🔹 Trick

Return subtree sum.
Check if subtree sum equals target.

---

### 🔹 Solution

```python
class Solution:
    def sumSubtree(self, root, target):
        self.found = False
        
        def dfs(node):
            if not node:
                return 0
            
            l = dfs(node.left)
            r = dfs(node.right)
            
            total = node.data + l + r
            
            if total == target:
                self.found = True
            
            return total
        
        dfs(root)
        return self.found
```

# 4️⃣ Count Nodes Equal to Sum of Descendants (Leetcode 1973)

🔗 [https://leetcode.com/problems/count-nodes-equal-to-sum-of-descendants/](https://leetcode.com/problems/count-nodes-equal-to-sum-of-descendants/)

Type: Subtree aggregation + count

---

### 🔹 Trick

Return subtree sum.
Compare node value with left + right.

---

### 🔹 Solution

```python
class Solution:
    def equalToDescendants(self, root: Optional[TreeNode]) -> int:
        self.count = 0
        
        def dfs(node):
            if not node:
                return 0
            
            l = dfs(node.left)
            r = dfs(node.right)
            
            if node.val == l + r:
                self.count += 1
            
            return node.val + l + r
        
        dfs(root)
        return self.count
```
