# 4️⃣ Count Complete Tree Nodes (Optimized)

🔗 [https://leetcode.com/problems/count-complete-tree-nodes/](https://leetcode.com/problems/count-complete-tree-nodes/)

(Type: Optimized count using structure)

---

### 🔹 Trick

If left height == right height
→ perfect tree → use formula:

```python
(1 << height) - 1
```

---

### 🔹 Solution

```python
class Solution:
    def countNodes(self, root):
        if not root:
            return 0
        
        def leftHeight(node):
            h = 0
            while node:
                h += 1
                node = node.left
            return h
        
        def rightHeight(node):
            h = 0
            while node:
                h += 1
                node = node.right
            return h
        
        lh = leftHeight(root)
        rh = rightHeight(root)
        
        if lh == rh:
            return (1 << lh) - 1
        
        return 1 + self.countNodes(root.left) + self.countNodes(root.right)
```

# 5️⃣ Count Good Nodes (Leetcode 1448)

🔗 [https://leetcode.com/problems/count-good-nodes-in-binary-tree/](https://leetcode.com/problems/count-good-nodes-in-binary-tree/)

(Type: Path condition based count)

---

### 🔹 Trick

Pass max so far downwards.

---

### 🔹 Solution (Your Version)

```python
class Solution:
    def goodNodes(self, node: TreeNode) -> int:

        def dfs(root, max_val):
            if not root:
                return 0

            if root.val >= max_val:
                count = 1
            else:
                count = 0

            max_val = max(max_val, root.val)

            l = dfs(root.left, max_val)
            r = dfs(root.right, max_val)

            return count + l + r

        return dfs(node, node.val)
```

# 6️⃣ Count Nodes in Range (BST)

(Type: BST Pruning)

---

### 🔹 Trick

If node.val < low → go right only
If node.val > high → go left only

---

### 🔹 Solution

```python
def countInRange(root, low, high):
    if not root:
        return 0
    
    if root.val < low:
        return countInRange(root.right, low, high)
    elif root.val > high:
        return countInRange(root.left, low, high)
    else:
        return 1 + countInRange(root.left, low, high) + countInRange(root.right, low, high)
```

# 7️⃣ Count Univalue Subtrees (Leetcode 250)

🔗 [https://leetcode.com/problems/count-univalue-subtrees/](https://leetcode.com/problems/count-univalue-subtrees/)

(Type: Condition + subtree validation)

---

### 🔹 Trick

Return whether subtree is univalue.

---

### 🔹 Solution

```python
class Solution:
    def countUnivalSubtrees(self, root: Optional[TreeNode]) -> int:
        self.count = 0
        
        def dfs(node):
            if not node:
                return True
            
            left = dfs(node.left)
            right = dfs(node.right)
            
            if not left or not right:
                return False
            
            if node.left and node.left.val != node.val:
                return False
            if node.right and node.right.val != node.val:
                return False
            
            self.count += 1
            return True
        
        dfs(root)
        return self.count
```
