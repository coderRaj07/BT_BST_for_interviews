# 1️⃣ Max Sum Root → Leaf (GFG)

🔗 [https://practice.geeksforgeeks.org/problems/root-to-leaf-path-sum/1](https://practice.geeksforgeeks.org/problems/root-to-leaf-path-sum/1)

### 🔹 Trick

Leaf trigger + choose max branch.

---

### 🔹 Solution (Your Version)

```python
class Solution:
    def maxPathSum(self, root):
        if not root:
            return float('-inf')
        if not root.left and not root.right:
            return root.data
        
        l_max_sum = self.maxPathSum(root.left)
        r_max_sum = self.maxPathSum(root.right)
        
        return root.data + max(l_max_sum,r_max_sum)
```

# 2️⃣ Path Sum (Leetcode 112)

🔗 [https://leetcode.com/problems/path-sum/](https://leetcode.com/problems/path-sum/)

### 🔹 Trick

Subtract while going down.
Check only at leaf.

---

### 🔹 Solution (Your Version)

```python
class Solution:
    def hasPathSum(self, root: Optional[TreeNode], targetSum: int) -> bool:
        if not root:
            return False
        
        if not root.left and not root.right:
            return root.val==targetSum 
        
        l = self.hasPathSum(root.left, targetSum-root.val)
        r = self.hasPathSum(root.right, targetSum-root.val)

        return l or r
```

# 3️⃣ Path Sum II (Leetcode 113)

🔗 [https://leetcode.com/problems/path-sum-ii/](https://leetcode.com/problems/path-sum-ii/)

### 🔹 Trick

Root → Leaf
Store path
Backtrack

---

### 🔹 Solution (Your Version)

```python
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> List[List[int]]:
        res = []

        def dfs(node, curr, target):
            if not node:
                return

            curr.append(node.val)

            if not node.left and not node.right and target == node.val:
                res.append(curr[:])

            dfs(node.left, curr, target - node.val)
            dfs(node.right, curr, target - node.val)

            curr.pop()

        dfs(root, [], targetSum)
        return res
```
### or

```python
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> List[List[int]]:
        self.res = []

        def dfs(node, curr, total):
            if not node:
                return []
            
            curr.append(node.val)
            total+=node.val

            if not node.left and not node.right:
                if total == targetSum:
                    self.res.append(curr[:])
            
            dfs(node.left, curr, total)
            dfs(node.right, curr, total)

            curr.pop()

        dfs(root, [], 0)
        return self.res
```

# 4️⃣ Count Root → Leaf Paths

🔗 [https://practice.geeksforgeeks.org/problems/count-leaves-in-binary-tree/1](https://practice.geeksforgeeks.org/problems/count-leaves-in-binary-tree/1)

### 🔹 Trick

Each leaf = 1 path.

---

### 🔹 Solution

```python
def countPaths(root):
    if not root:
        return 0
    if not root.left and not root.right:
        return 1
    return countPaths(root.left) + countPaths(root.right)
```

# 6️⃣ Sum Root to Leaf Numbers (Leetcode 129)

🔗 [https://leetcode.com/problems/sum-root-to-leaf-numbers/](https://leetcode.com/problems/sum-root-to-leaf-numbers/)

### 🔹 Trick

Carry number while going down.
Multiply by 10.

---

### 🔹 Solution

```python
class Solution:
    def sumNumbers(self, root: Optional[TreeNode]) -> int:
        def dfs(node, curr):
            if not node:
                return 0
            
            curr = curr * 10 + node.val
            
            if not node.left and not node.right:
                return curr
            
            return dfs(node.left, curr) + dfs(node.right, curr)
        
        return dfs(root, 0)
```
