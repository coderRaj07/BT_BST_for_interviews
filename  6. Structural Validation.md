# 1️⃣ Same Tree (Leetcode 100)

🔗 [https://leetcode.com/problems/same-tree/](https://leetcode.com/problems/same-tree/)

Type: Structure comparison

---

### 🔹 Trick

Both must be null
Both must exist
Values must match
Recurse on left & right

---

### 🔹 Solution

```python id="s6_1"
class Solution:
    def isSameTree(self, p: Optional[TreeNode], q: Optional[TreeNode]) -> bool:
        if not p and not q:
            return True
        if not p or not q:
            return False
        if p.val != q.val:
            return False
        
        return self.isSameTree(p.left, q.left) and self.isSameTree(p.right, q.right)
```

# 2️⃣ Symmetric Tree (Leetcode 101)

🔗 [https://leetcode.com/problems/symmetric-tree/](https://leetcode.com/problems/symmetric-tree/)

Type: Mirror comparison

---

### 🔹 Trick

Compare:

* left of A with right of B
* right of A with left of B

---

### 🔹 Solution (Your Version)

```python id="s6_2"
class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        def dfs(a,b):
            if not a and not b:
                return True
            if not a or not b:
                return False

            return  a.val==b.val and dfs(a.left, b.right) and dfs(b.left,a.right)
        
        return dfs(root.left, root.right)
```

# 3️⃣ Balanced Binary Tree (Leetcode 110)

🔗 [https://leetcode.com/problems/balanced-binary-tree/](https://leetcode.com/problems/balanced-binary-tree/)

Type: Height validation

---

### 🔹 Trick

Return -1 if unbalanced
Otherwise return height

---

### 🔹 Solution (Your Version)

```python id="s6_3"
class Solution:
    def isBalanced(self, node: Optional[TreeNode]) -> bool:

        def dfs_height(root):
            if not root:
                return 0
            
            lh = dfs_height(root.left)
            if lh==-1:
                return -1

            rh = dfs_height(root.right)
            if rh==-1:
                return -1

            if(abs(lh-rh)>1):
                return -1

            return 1+max(lh,rh)    

        return dfs_height(node)!=-1
```

# 4️⃣ Invert Binary Tree (Leetcode 226)

🔗 [https://leetcode.com/problems/invert-binary-tree/](https://leetcode.com/problems/invert-binary-tree/)

---

### 🔹 Trick

Swap left & right at every node.

---

### 🔹 Solution

```python id="s6_4"
class Solution:
    def invertTree(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        if not root:
            return None
        
        root.left, root.right = root.right, root.left
        
        self.invertTree(root.left)
        self.invertTree(root.right)
        
        return root
```

# 5️⃣ Flatten Binary Tree (Leetcode 114)

🔗 [https://leetcode.com/problems/flatten-binary-tree-to-linked-list/](https://leetcode.com/problems/flatten-binary-tree-to-linked-list/)

Type: Pointer restructuring

---

### 🔹 Trick

Reverse preorder:
Right → Left → Root
Use prev pointer

---

### 🔹 Solution (Your Version)

```python id="s6_5"
class Solution:
    def flatten(self, root: Optional[TreeNode]) -> None:
        self.prev = None
        
        def dfs(node):
            if not node:
                return
            dfs(node.right)
            dfs(node.left)
            node.right = self.prev
            node.left = None
            self.prev = node
        
        dfs(root)
        return self.prev
```

# 6️⃣ Maximum Width of Binary Tree (Leetcode 662)

🔗 [https://leetcode.com/problems/maximum-width-of-binary-tree/](https://leetcode.com/problems/maximum-width-of-binary-tree/)

Type: Level order + indexing

---

### 🔹 Trick

Assign indices:
Left = 2*i
Right = 2*i + 1

Width = last - first + 1

---

### 🔹 Solution (Your Version)

```python id="s6_6"
class Solution:
    def widthOfBinaryTree(self, node: Optional[TreeNode]) -> int:
        q = deque([(node, 0)])
        maxi = 0
        while q:
            size = len(q)
            _, first = q[0]
            _, last = q[-1]
            
            width = last - first + 1
            maxi = max(maxi, width)

            for _ in range(size):
                node, idx = q.popleft()

                if node.left:
                    q.append((node.left, 2*idx))
                if node.right:
                    q.append((node.right, 2*idx+1))
        return maxi
```

# 7️⃣ Validate BST (Leetcode 98)

🔗 [https://leetcode.com/problems/validate-binary-search-tree/](https://leetcode.com/problems/validate-binary-search-tree/)

Type: BST validation

---

### 🔹 Trick

Use range (low, high)

---

### 🔹 Solution

```python id="s6_7"
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        def dfs(node, low, high):
            if not node:
                return True
            
            if not (low < node.val < high):
                return False
            
            return dfs(node.left, low, node.val) and dfs(node.right, node.val, high)
        
        return dfs(root, float('-inf'), float('inf'))
```
